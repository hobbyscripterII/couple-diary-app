## ⚡ 프로젝트 후기
**좋았던 점**
- 좋은 팀원분들을 만나 덕분에 재미있게 진행했다.
- 개인 프로젝트만 하다가 협업을 해보니 재밌었고 커뮤니케이션의 중요성을 깨달았다.
- 서로 팀 간의 의견을 조율하며 프로젝트를 진행하는 게 재밌었다.
- 백엔드, 프론트엔드 두 팀으로 나눠 작업하고 결과물을 하나로 합치는 작업이 재밌었다. <br>

**아쉬웠던 점**
- 배운 것을 토대로 구현하려고 하니 쉽지 않았다.
- 소통의 중요성을 깨달았다.
